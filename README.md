# Distance App

A simple Python application for logging session environment data using a terminal or a Streamlit web interface.

---

## 🚀 Features

- Record measured distance
- Record wind speed
- Record wind direction
- Record temperature
- Record location
- Add custom notes
- Save all data to a CSV file
- View recent entries in a web UI
